# Qhamuka High School — Front-End

Responsive, accessible front-end for Qhamuka High School, built with HTML templates,
Tailwind CSS, and vanilla JavaScript. Structured for a Flask/Jinja2 blueprint backend.

## Backend setup & run

```bash
python -m venv venv
source venv/bin/activate          # Windows: venv\Scripts\activate
pip install -r requirements.txt
python seed.py
python run.py
# -> http://127.0.0.1:5000
```
